ClaudeCodeNotify

Desktop notifications when Claude Code needs you — one keystroke back to your terminal.

Download for macOS ⭐ Star on GitHub

macOS 13+ · Apple Silicon · free & open source

It’s a notifier, not a gatekeeper. It doesn’t block tools or decide permissions — you still approve or deny in the terminal. It just makes sure you notice, and gets you there fast.

See it in action

You prompt, Claude works, you go do something else. The moment Claude needs you, a card pops up — and Enter brings your terminal straight back.

  • 🔔Floating cards when Claude asks permission, goes idle or finishes
  • ↵Enter focuses the exact app — Ghostty, iTerm, Terminal, Cursor, VS Code
  • ⚡Live usage bars — 5-hour & weekly limits, color-coded with reset countdown

Download

Option 1: Homebrew (Recommended)

The easiest way to install. It bypasses Gatekeeper warnings automatically:

brew install narlei/tap/claudecodenotify

Option 2: Manual Download. Get the latest drag-to-Applications .dmg from GitHub Releases.

Download latest .dmg

First launch (unsigned app)

The app is ad-hoc signed (no paid Apple account), so Gatekeeper will warn you the first time. Either right-click → Open, or clear the quarantine flag:

xattr -dr com.apple.quarantine /Applications/ClaudeCodeNotify.app
Next:
  1. Open Terminal
  2. Press ⌘V, then Enter

Requirements: macOS 13+, Apple Silicon.
